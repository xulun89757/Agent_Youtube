# YouTube Agent

## 项目简介

自动监控 YouTube 财经频道。

发现新视频后：

1. 获取最新视频
2. 使用 Gemini 分析
3. 自动发送到飞书
4. 防止重复推送

---

## 技术栈

- Node.js
- GitHub Actions
- Gemini API
- YouTube Data API
- 飞书机器人
